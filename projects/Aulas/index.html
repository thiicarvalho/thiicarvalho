Precedencia de execuçao

O JavaScript é uma linguagem usada principalmente para interaçao com os elementos HTML. O ideal é que a ligaçao para o arquivo JS seja feita somente no final do arquivo, visto que para que haja uma interaçao, é preciso que o elemento seja carregado na pagina.

Variaveis

Variaveis sao espaços dentro da memoria do dispositivo que usamos para armazenar dados temporariamente. Essas variaveis podem ser de diversos tipos: strings (responsaveis pelo armazenamento de textos), numbers (responsaveis pelo armazenamento de numeros inteiros ou fracionarios), boolean (responsaveis pelo armazenamento de resultados "true" ou "false").
Para declarar uma variavel existem algumas regras a serem seguidas.

  • Nao podem ser iniciadas com numeros, apenas letras ou _
  • Nao podem ser utilizados caracteres especiais como "ç", "^", "~"...
  • Nao podem ser iguais as palavras usadas na linguagem, como "var", "let", "get", etc
  • A linguagem diferencia maiusculas de minusculas, entao "curso" é diferente de "Curso", que por sua vez é diferente de "CURSO"

    • Sao elementos que permitem o armazenamento de dados dentro da memoria do computador. Para os nomes dos identificadores, maiusculas e minusculas fazem diferença.
    Variaveis devem conter nomes coerentes, ou seja, quando se tratar de salarios por exemplo, utilize o nome salario para a variavel.
    Toda vez que o sinal de = aparecer no JS, ele significa "recebe".
    Por exemplo: variavel1 = elemento1, ou seja variavel1 recebe elemento1.

    Ao declarar uma variavel, é recomendado usarmos o termo "var", visto que tanto para o JS quanto para quem vai trabalhar com o codigo, é mais facil interpretar que o termo var diz respeito à criaçao de uma variavel. Lembrando que nao devemos utilizar espaços no nome da variavel.

Null e Undefined

Null representa a ausencia intencional de um valor; ou seja a variavel sera declarada mas indicaremos que ela faz referencia a uma posiçao dentro do nosso codigo.
Ja undefined significa que a variavel nao possui valor inexistente ou nao atribuido. Uma variavel declarada sera undefined até o momento em que atribuimos um valor a ela.

Trabalhando com funçoes

Para trabalhar a localizaçao de tipos de dados usamos o

Operadores


Alguns exemplos sao os aritiméticos, operadores de atribuiçao, relacionais, logicos e ternarios.
Aprender a como trabalhar com operadores é necessario pois é com eles que vamos fazer calculos dentro da linguagem.
Operadores aritiméticos sao:
+ (soma),
-(subtraçao),
*(multiplicaçao),
/(divisao),
%(resto de divisao) e
**(potência). Sendo que o operador de % significa que é o resultado do resto de uma divisao inteira, quando é preciso inserir uma virgula nos paramos e levamo em conta o resto da divisao.
A ordem de precedencia é:
( )
**
*, / ou %
e por fim + e -.
Caso mais de um operador de precedencia apareça na mesma linha, iremos calcular da esquerda para a direita.
Operador de atribuiçao é quando você define o valor de uma variavel.
Operador de incremento: quando você precisa inserir um antecessor (-1) ou sucessor (+1), é possivel usar o operador de incremento. Ou seja, variavel++ ou variavel--. Por exemplo: n=n+1 pode ser substituido por n++ e n=n-1 pode ser substituido por n--.

Operadores relacionais sao os sinais de: <(menor), >(maior), <=(menor ou igual), >=(maior ou igual), ==(igual), !=(diferente), ===(operédor de identidade, ou seja o tipo e o valor iserido) e !==(desigual).
O resultado desses operadores sera sempre um valor boalean (true ou false)
Operadores logicos: !(negaçao), &&(conjunçao também conhecido como e) e ||(disjunçao, também conhecido como ou).
A ordem de precedência dos operadores é: Aritiméticos > Relacionais > Logicos (na seguinte ordem: ! > && > ||).

Operador ternario: ? e : dentro da mesma expressao. Por exemplo: teste ? true : false

DOM (Document Object Model)

Modelo para acesso aos objetos dentro do documento. E importante saber usar a arvore DOM. definindo quais sao os elementos pais e filhos.
Usado para acessar todos os elementos dentro do navegador, como por exemplo: tag, id, class, name e seletor.
Até o momento dessa aula o unico que funcionou é o 'getElementsByTagName', por este motivo vou usar o 'querySelector' sempre que possivel.

Condiçoes em JS

Condiçoes é aplicar açoes que serao executadas de acordo com a interatividade do usuario fazendo com que o JS reaja de uma maneira ou de outra.

IF

Condiçao simples, possui somente a condiçao true
Condiçao composta possui os resultados true e false

Condiçoes Aninhadas

Condiçoes que podem conter outras condiçoes dentro delas.

ELSEIF

Representa qualquer valor que nao se enquadra no if e else.

Condiçoes Multiplas

switch (Expressao){
case 1()
break (obrogatorio)
case 2()
break (obrogatorio)
etc... }
Usado para dados pontuais, ou seja para casos especificos. Funciona apenas com numeros inteiros e textos.
